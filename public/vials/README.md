# Vial images

Drop PNGs here named `<sku>.png` (lowercase). Each product in `src/data/products.js` has a `sku` field — the filename must match it exactly (lowercased).

## Expected filenames

```
op-glp3-10mg.png
op-glp3-20mg.png
op-glp3-10mg-kit.png
op-glp3-20mg-kit.png
op-bpc-5mg.png
op-bpc-5mg-kit.png
op-bpc-10mg.png
op-bpc-10mg-kit.png
op-tb500-5mg.png
op-tb500-5mg-kit.png
op-tb500-10mg.png
op-tb500-10mg-kit.png
op-combo-70mg.png
op-combo-70mg-kit.png
op-ipa-5mg.png
op-ipa-5mg-kit.png
op-hgh-10iu-kit.png
op-mt2-5mg.png
op-mt2-5mg-kit.png
op-nad-500mg.png
op-nad-500mg-kit.png
```

## Format
- PNG with transparent background (ideal — works on any page bg)
- Square aspect ratio recommended
- Minimum ~600×600 px for crisp rendering on hero sections
- Filename must be lowercase

Any SKU without a matching image will fall back to the placeholder SVG.
